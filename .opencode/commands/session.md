# Session Commands

## session status
Show session status.
```bash
cli-anything-blender session status
```

## session undo
Undo the last operation.
```bash
cli-anything-blender session undo
```

## session redo
Redo the last undone operation.
```bash
cli-anything-blender session redo
```

## session history
Show undo history.
```bash
cli-anything-blender session history
```
